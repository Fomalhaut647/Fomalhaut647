### 🤖 Financial & Accounting Workflow Automation AI Agent

I'm currently working on an AI Agent project for automating financial and accounting workflows for small and medium-sized enterprises.

Many companies already have financial software, but before data can enter these systems, there is still a large amount of messy manual work: invoice photos, contract PDFs, bank statements, non-standard Excel files, and business ledgers need to be extracted, cleaned, reconciled, summarized, split, and then converted into reports or accounting vouchers.

I hope to build an intelligent preprocessing layer between raw business data and financial systems:

- Supports input from invoices, contracts, Excel files, bank statements, and other raw materials
- Automatically performs structured extraction, data cleaning, reconciliation, and calculation
- Generates reviewable Excel files, reports, or import-ready formats for financial systems
- Emphasizes auditability, traceability, and reproducibility
- Allows humans to review and correct results at key steps

This project is still under active iteration. I'm validating its demand boundaries, MVP priorities, and delivery methods through enterprise internship experience, industry interviews, and observation of real financial workflows.
